# Spotify Clone (HTML & CSS)

This is my **first front-end project**, where I tried to recreate the basic UI of Spotify using only **HTML and CSS**.

The goal of this project was to practice **layout design, styling, and basic web development concepts**.

---

## Project Overview

This project replicates the **visual layout of Spotify**, including elements like:

* Navigation sidebar
* Music cards / playlists
* Basic layout structure
* Styling similar to the Spotify interface

⚠️ This project focuses only on the **UI design** and does not include music playback functionality.

---

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Basic responsive layout techniques

---

## What I Learned

Through this project, I learned:

* Structuring web pages with HTML
* Styling layouts using CSS
* Using Flexbox for alignment
* Creating UI components
* Understanding how real websites are structured
